# 🌍 COVID-19 Exploratory Data Analysis – Data Visualization & EDA

## 📌 Project Overview
This project performs **Exploratory Data Analysis (EDA)** on a country-wise COVID-19 dataset using Python.

The objective was to:

- Perform data cleaning and preprocessing  
- Analyze global pandemic trends  
- Create visualizations for trends, correlations, and outliers  
- Identify key regional patterns in COVID-19 cases  
- Derive actionable insights from the data  

---

## 🔍 Analysis Performed

### Data Cleaning
- Checked dataset structure and missing values  
- Renamed columns for easier analysis  
- Ensured numerical columns were correctly formatted  
- Handled missing or inconsistent data where necessary  

### Trend Analysis
- Top countries with highest confirmed cases  
- Countries with highest deaths  
- Weekly growth in cases across countries  

### Correlation & Outlier Detection
- Correlation heatmap between pandemic indicators  
- Boxplot analysis to detect extreme case values  

### Distribution Analysis
- Global case distribution (Confirmed, Recovered, Deaths, Active)  
- Case distribution across WHO regions  
- Top country contribution to global cases  

---

## 📊 Key Insights
- A small number of countries contribute to a large share of global COVID-19 cases.  
- Confirmed cases show a strong positive relationship with deaths.  
- Some countries experienced rapid weekly case growth indicating emerging outbreaks.  
- Pandemic impact varies significantly across WHO regions.  
- Extreme case counts highlight countries requiring urgent intervention.
